# Сервис Кредитного Скоринга 

Этот проект представляет собой веб-приложение для оценки кредитоспособности с использованием модели машинного обучения. Он состоит из внешнего интерфейса, построенного с помощью Streamlit, и бэкенд-API, построенного с помощью FastAPI.

## Структура Проекта

*  `app.py`: Приложение Streamlit, реализующее пользовательский интерфейс. Предоставляет веб-форму для ввода данных пользователями.
*  `service.py`: Приложение FastAPI, реализующее бэкенд. Предоставляет конечную точку API для расчета кредитного скоринга.
*  `models/credit_scoring_pipeline.pkl`: Предварительно обученный конвейер модели машинного обучения, используемый для прогнозирования.
*  `requirements.txt`: Список всех зависимостей Python, необходимых для этого проекта.

## Предварительные условия

*  Python 3.8+
*  pip

## Установка

1.  Клонируйте репозиторий:
    ```sh
    git clone https://github.com/your-username/creditscoring-personal.git
    cd creditscoring-personal/Submodules/credit_scoring
    ```

2.  Установите необходимые зависимости:
    ```sh
    pip install -r requirements.txt
    ```

## Использование

Для запуска приложения необходимо запустить как бэкенд-сервис, так и клиентское приложение.

1.  **Запуск бэкенда FastAPI:**
    Откройте терминал и выполните следующую команду. Это запустит бэкенд-сервер по адресу `http://localhost:8000`.
    ```sh
    uvicorn service:app --reload
    ```

2.  **Запуск клиентского приложения Streamlit:**
    Откройте второй терминал и выполните следующую команду. Это откроет веб-приложение в вашем браузере.
    ```sh
    streamlit run app.py
    ```

Теперь вы можете заполнить форму в веб-интерфейсе и получить результат кредитного скоринга.

## API Эндпоинт

*  `POST /score`
    *  Принимает JSON-объект с данными клиента.
    *  Возвращает JSON-объект с решением по кредиту (`"approved": true/false`).
