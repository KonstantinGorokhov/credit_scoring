## Кредитный скоринг

Проект представляет собой демонстрационный сервис кредитного скоринга, реализующий полный цикл взаимодействия пользователя с ML-системой: от ввода данных клиента через веб-интерфейс до получения скорингового результата через API.

Архитектура включает:
- **Frontend на Streamlit** — форма ввода данных клиента и отображение результата скоринга
- **Backend на FastAPI** — REST API для приёма данных и расчёта скорингового решения
- **Модуль генерации признаков** — формирование скрытых параметров, используемых в расчёте


### Сборка и запуск контейнеров

В корневой директории проекта выполните команду:

```bash
    docker-compose up --build
```

### Анализ минимальных критериев для выдачи кредита

В `backend/models` находится `jupyter notebook` `credit_analysis_min_features.ipynb` с анализом.

### Обучение модели

В той же директории находится `model.ipynb` с обучением 4 моделей и их метриками.
