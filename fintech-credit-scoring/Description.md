# Описание набора данных

*  https://www.kaggle.com/c/fintech-credit-scoring/data

В данном документе представлено описание набора данных, используемого для задачи кредитного скоринга. Данные включают анкетную информацию о заявителях и сведения о случаях дефолта.

*** 

## Структура данных

*   `application_info.csv` — Анкетные данные заявителей.
*   `default_flg.csv` — Сведения о дефолтах.

*** 

## Описание полей

|   | Наименование     | Описание                                                                  |
|---|------------------|---------------------------------------------------------------------------|
| ❌ | `id`               | Анонимизированный идентификатор заявителя                              |
| ❌ | `application_dt`   | Дата подачи заявки                                                     |
| ❌ | `sample_cd`        | Категория выборки (обучающая/тестовая)                                 |
| ✅ | `education_cd`     | Уровень образования                                                    |
| ❌ | `gender_cd`        | Пол заявителя                                                          |
| ✅ | `age`              | Возраст заявителя                                                      |
| ✅ | `car_own_flg`      | Флаг наличия автомобиля                                                |
| ✅ | `car_type_flg`     | Флаг наличия автомобиля иностранного производства                      |
| ✅ | `appl_rej_cnt`     | Количество отклоненных заявок в прошлом                                |
| ✅ | `good_work_flg`    | Признак "хорошей" работы (стабильность, репутация работодателя)        |
| ✅ | `Score_bki`        | Скоринговый балл по данным из БКИ (Бюро кредитных историй)             |
| ✅ | `out_request_cnt`  | Количество запросов в БКИ                                              |
| ✅ | `region_rating`    | Рейтинг региона проживания                                             |
| ✅ | `home_address_cd`  | Категория домашнего адреса                                             |
| ✅ | `work_address_cd`  | Категория рабочего адреса                                              |
| ✅ | `income`           | Ежемесячный доход заявителя                                            |
| ✅ | `SNA`              | Параметр, характеризующий социальные связи заявителя                   |
| ✅ | `first_time_cd`    | Давность первичной информации о заявителе                              |
| ✅ | `Air_flg`          | Флаг наличия заграничного паспорта                                     |
| ✅ | `default_flg`      | Флаг дефолта по кредиту (1 - был дефолт, 0 - не было)                  |
